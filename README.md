# play-nextcloud
### Intro
    - Gần giống với Google Drive.
    - Ngoài ra có nhiều chức năng hay ho hơn.

### Có hai cách triển khai
1. Triển khai cách 1:
- Đứng sau Reverse Proxy.
- ví dụ: version 2
2. Triển khai cách 2:
- Trong quá trình khiển khai có luôn proxy và mã hóa ssl
- ví dụ: version 6
